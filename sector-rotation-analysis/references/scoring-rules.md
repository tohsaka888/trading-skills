# 评分规则

## 1. 输出结构
每个板块必须同时输出：
- 短线分 `0-100`
- 长线分 `0-100`
- 热度状态
- 流动性状态
- 是否禁止追高
- 短线动作
- 长线动作

## 2. 短线分
短线分 = 资金动量 + 板块趋势与15分钟结构 + 流动性质量 + 新闻催化 + 风险收益比与执行质量 + 追高惩罚

### 2.1 资金动量（30）
- 今日、5日、10日行业资金流都靠前，且排序稳定：`25-30`
- 今日强、5日和10日同步改善：`15-20`
- 仅单日脉冲或明显转弱：`0-10`

### 2.2 板块趋势与 15 分钟结构（25）
- 日线强、15 分钟结构稳，回踩承接清晰：`20-25`
- 日线偏强但分时波动大：`10-15`
- 冲高回落或短线破位：`0-5`

### 2.3 流动性质量（20）
- 成交量与成交额温和放大，量价匹配，换手率抬升但不过热：`15-20`
- 量能改善但持续性一般，或换手率偏高需观察：`8-14`
- 缩量上涨、爆量滞涨或流动性失真：`0-7`

### 2.4 新闻催化（15）
- 近 7 天存在高可信度政策或产业催化，且与资金方向一致：`12-15`
- 有事件但强度一般或偏中性：`6-10`
- 催化不足或存在明确利空：`0-5`

### 2.5 风险收益比与执行质量（10）
- 拥挤度可控，回撤预期清晰，执行窗口明确：`8-10`
- 收益风险比一般，需等待更好介入点：`4-7`
- 拥挤或不确定性高，执行难度大：`0-3`

### 2.6 追高惩罚
- 不触发：`0`
- 满足 2 条追高信号：`-10`
- 满足 3 条及以上追高信号：`-20`

## 3. 长线分
长线分 = 中期资金持续性 + 周线/月线趋势延续性 + 政策与产业景气 + 流动性健康度 + 风险控制

### 3.1 中期资金持续性（20）
- 5日与10日持续靠前，资金曲线稳定：`15-20`
- 仅 5 日强，10 日一般：`8-14`
- 中期资金不持续：`0-7`

### 3.2 周线/月线趋势延续性（25）
- 周线和月线同向上行，回撤可控：`19-25`
- 仅周线偏强，月线未完全确认：`10-18`
- 周月结构走弱或破坏：`0-9`

### 3.3 政策与产业景气（20）
- 存在高可信度政策支持、景气上行或订单扩张证据：`15-20`
- 产业逻辑中性，催化不连续：`8-14`
- 产业逻辑承压或政策不利：`0-7`

### 3.4 流动性健康度（25）
- 量能稳定、成交额充足、换手率健康，资金承接持续：`19-25`
- 流动性一般，仍需观察资金承接：`10-18`
- 成交结构脆弱或过热失真：`0-9`

### 3.5 风险控制（10）
- 政策、供需、拥挤度风险可控，回撤结构健康：`8-10`
- 有部分风险但仍可跟踪：`4-7`
- 风险集中暴露：`0-3`

## 4. 追高判定
满足以下任意 `2` 条，即标记“禁止追高”：
1. 板块近 `5` 个交易日涨幅 `> 15%`
2. 板块最新收盘偏离 `20` 日均价 `> 12%`
3. 最新成交额或成交量显著高于 `20` 日均值且 `15` 分钟出现尾盘转弱
4. 今日资金流强，但 `5日` 和 `10日` 资金流都未进入前 `20`

## 5. 潜力板块判定
同时满足以下条件，才可进入“潜力观察”：
1. 长线分 `>= 70`
2. 短线分 `< 60`
3. 新闻面净判断为偏多
4. 周线和月线结构未破坏
5. 流动性边际改善且不过热

## 6. 热度状态
- `高热`：短线分 `>= 75`
- `升温`：短线分 `60-74`
- `蓄势`：短线分 `45-59`
- `降温`：短线分 `< 45`

## 7. 流动性状态
- `放量但可控`：成交量与成交额改善，换手率健康
- `温和改善`：量能边际转强，但尚未全面确认
- `平稳`：量能与换手率变化有限
- `过热`：放量过猛、尾盘转弱或换手率异常
- `偏弱`：缩量或承接不足

## 8. 动作映射

### 短线动作
- `80-100`：`积极参与`
- `65-79`：`回踩参与`
- `45-64`：`观察`
- `0-44`：`回避`

### 长线动作
- `80-100`：`逐步配置`
- `65-79`：`回调配置`
- `45-64`：`观察`
- `0-44`：`回避`

### 追高门控
- 若标记“禁止追高”，短线动作最高只能为 `回踩参与`

## 9. 置信度
- `高`：核心板块数据、资金数据、新闻数据都完整
- `中`：任一模块部分缺失，但不影响主结论
- `低`：两类及以上关键数据缺失，或新闻仅有低可信度来源
