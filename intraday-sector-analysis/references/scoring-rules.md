# 评分规则

## 1. 输出结构
每个板块必须同时输出：
- `盘中关注分` `0-100`
- `做T适配分` `0-100`
- 热度判断
- 是否禁止追高
- 是否适合做T
- 建议动作
- 触发条件
- 失效条件

## 2. 盘中关注分
盘中关注分 = 今日资金强度 + 中短期资金持续性 + 异动活跃度 + 实时行情强度 + 分时结构质量 + 日线技术位置 - 过热惩罚

### 2.1 今日资金强度（30）
- 今日行业资金流进入前 `5` 且净流入力度显著：`24-30`
- 今日行业资金流进入前 `10`：`16-23`
- 今日有流入但排名一般：`8-15`
- 今日不强或明显转弱：`0-7`

### 2.2 中短期资金持续性（15）
- `5日` 和 `10日` 同时靠前：`12-15`
- 仅 `5日` 靠前：`7-11`
- 仅单日强脉冲：`0-6`

### 2.3 异动活跃度（15）
- 当日异动次数显著靠前，说明资金持续点火：`12-15`
- 有异动但活跃度一般：`6-11`
- 异动不明显：`0-5`

### 2.4 实时行情强度（15）
- 实时涨幅、成交额、振幅和承接都强，处于日内强势区：`12-15`
- 实时强度中性偏强：`6-11`
- 冲高回落或承接不足：`0-5`

### 2.5 分时结构质量（15）
- 开盘强、回落浅、午后再强化，当前位置接近日内强势区：`12-15`
- 结构可交易但反复较大：`6-11`
- 冲高回落明显或分时失真：`0-5`

### 2.6 日线技术位置（10）
- `5/10/20` 日均线多头，近 `3` 根日K健康，位置不算过热：`8-10`
- 日线偏强但斜率一般或已偏离中枢：`4-7`
- 日线结构不稳或追高风险高：`0-3`

### 2.7 过热惩罚
- 不触发：`0`
- 触发 `2` 条风险门控：`-10`
- 触发 `3` 条及以上风险门控：`-20`

## 3. 做T适配分
做T适配分 = 流动性质量 + 分时波动可交易性 + 回落承接 + 技术中枢稳定性 + 拥挤度调整

### 3.1 流动性质量（25）
- 成交额充足、换手健康、进出流畅：`19-25`
- 流动性一般但可以执行：`10-18`
- 流动性弱或失真：`0-9`

### 3.2 分时波动可交易性（25）
- 日内有明确高低点和可重复波段，适合低吸高抛：`19-25`
- 波动存在但节奏一般：`10-18`
- 单边拉升或单边下杀，不适合做T：`0-9`

### 3.3 回落承接（20）
- 回落后快速企稳并重新站回强势区：`15-20`
- 有承接但力度一般：`8-14`
- 回落后承接差：`0-7`

### 3.4 技术中枢稳定性（15）
- 围绕日线中枢运行，回撤不破位：`12-15`
- 结构尚可但波动较大：`6-11`
- 技术中枢不稳：`0-5`

### 3.5 拥挤度调整（15）
- 热度可控，做T不容易被情绪挤压：`11-15`
- 热度偏高但仍可控：`6-10`
- 拥挤度高，做T容易被单边波动打穿：`0-5`

## 4. 固定技术指标

### 日线
- `5/10/20` 日均线关系
- 近 `5` 日涨幅
- 最新价相对 `20` 日均线偏离
- 最近 `3` 根日K强弱

### 分时
- 开盘 `30` 分钟强弱
- 上午高点是否明显回吐
- 午后是否再创新高或再强化
- 当前价格位于日内高低区间的百分位
- 分时回撤是否被承接

## 5. 风险门控
满足以下任意 `2` 条，即标记“禁止追高”：
1. 近 `5` 日涨幅 `> 12%`
2. 最新价偏离 `20` 日均线 `> 8%`
3. 今日资金流强，但 `5日` 和 `10日` 资金流都未跟随
4. 分时冲高后回落到日内中位以下

若标记“禁止追高”，建议动作最高只能给 `回踩观察`。

## 6. 是否适合做T
- `做T适配分 >= 65` 且不处于 `09:35-10:30` 限制时段：`是`
- `做T适配分 50-64`：`一般，只能轻仓低吸高抛`
- `做T适配分 < 50` 或处于限制时段：`否`

## 7. 动作映射
- `盘中关注分 >= 80` 且 `做T适配分 < 60`：`顺势跟随`
- `盘中关注分 >= 80` 且 `做T适配分 >= 60`：`顺势持有，可做T`
- `盘中关注分 60-79` 且 `做T适配分 >= 65`：`做T优先`
- `盘中关注分 50-79` 且 `做T适配分 < 65`：`回踩观察`
- `盘中关注分 < 50`：`回避`

## 8. 热度判断
- `高热`：盘中关注分 `>= 80`
- `升温`：盘中关注分 `65-79`
- `跟踪`：盘中关注分 `50-64`
- `降温`：盘中关注分 `< 50`

## 9. 置信度
- `高`：资金流、异动、实时和分时数据完整
- `中`：任一模块有缺失，但不影响主结论
- `低`：分时或实时关键字段缺失，或深度分析样本不足
